Мультикурсор

Примеры для vs code

Alt+Click Вставить курсор
Ctrl+Alt+ ↑ / ↓ Вставить курсор сверху / снизу
Ctrl+U Отменить последнюю операцию курсора. Также можно кликнуть на курсор с зажатым alt для отмены

Убрать все лишнее

Примеры для vs code

Ctrl+K Z Включить/отключить zen mode. Режим, при котором убираются все панели и остается только вкладка с кодом
Ctrl+B Скрыть/Показать боковую панель
Ctrl+` Скрыть/Показать терминал

Проверить на переполнение контента

В dev tools пишем document.designMode = 'on' и редактируем контент
